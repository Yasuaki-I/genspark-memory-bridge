# Phase 1 タスクリスト（保存機能実装）

## Phase 1 の目的

Genspark チャット終了時に、Chrome 拡張のボタンを押すと、会話全文が Markdown 化され GitHub リポジトリに commit される。これを動く状態にする。

## 完成条件（受け入れ基準）

1. Genspark チャットページで拡張のポップアップを開き「保存」ボタンを押すと、当該チャットの全メッセージが Markdown ファイルとして GitHub リポジトリの `chats/` フォルダに commit される
2. 生成された Markdown は、ユーザー発話と AI 応答が判別可能で、見出し・コードブロック・リスト・太字が正しく保持されている
3. エラー発生時（GitHub API 失敗、DOM 取得失敗等）はポップアップに分かりやすいエラーメッセージが出る
