# Genspark Memory Bridge - Chrome Web Store 掲載情報

最終更新日: 2026-06-06

## 短い説明（132文字以内）

GensparkチャットをワンクリックでMarkdown化しGitHubに保存。新チャット開始時にコンテキストを自動検知し復元。チャットの記憶を永続化するメモリブリッジ。

## 詳細な説明

■ Genspark Memory Bridge とは

Gensparkでの長いチャットセッション、せっかく積み上げた文脈が
新しいチャットを開くたびにリセットされていませんか？

Genspark Memory Bridgeは、チャットの保存と復元を一体化した
Chrome拡張です。終了時にワンクリックで保存、次回開始時に
ワンクリックで復元。チャットの記憶を途切れさせません。

■ 主な機能

【保存フロー】
・Gensparkチャット全文をMarkdown形式に自動変換
・GitHubリポジトリにワンクリックで保存（自動commit）
・ファイル名は自動生成＋手動編集可能
・古いスナップショットの自動削除（最新版のみ保持）

【復元フロー】
・新しいチャットページを自動検知してバッジで通知
・コンテキスト（前回の文脈・方針・状態をまとめたファイル）をGitHubから取得
・ワンクリックでクリップボードにコピー → 貼り付けるだけ

■ こんな方におすすめ

・Gensparkで複数のプロジェクトを並行管理している方
・チャットの文脈を次回セッションに引き継ぎたい方
・AIとの議論内容をGitHubで版管理したい方
・Obsidian + Git で知識管理している方

■ 他ツールとの違い

・保存「だけ」ではなく、復元フローまで一体化
・保存先がGitHub → バージョン管理・差分確認が自然にできる
・Obsidian-Git連携でローカルナレッジベースと自動同期可能
・Genspark専用設計 → 軽量・高精度なDOM解析

■ プライバシー

・外部サーバーへのデータ送信は一切なし
・通信先はユーザー自身のGitHubリポジトリのみ
・トラッキング・広告SDK不使用
・すべての処理はブラウザ内で完結

■ セットアップ（3ステップ）

1. GitHubでPersonal Access Tokenを発行
2. 拡張の設定画面でユーザー名・リポジトリ名・PATを入力
3. 保存ボタンを押すだけ

■ 動作要件

・Google Chrome（最新版推奨）
・GitHubアカウント（無料プランで利用可能）
・対象ページ: https://www.genspark.ai/

■ サポート

ご質問・不具合報告はGitHubリポジトリのIssuesにて受け付けています。
